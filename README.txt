This program aims to transpose sheet music in real time. 
Currently, a preliminary staff - shifter has been implemented. To see a demo:

1. Navigate to 'demo' folder
2. Run 'chmod +x demo.sh'
3. Run './demo.sh'
4. Enter name of outputfile (following promt)
5. Enter path of a score image (png). Some images can be found in 'demo/data'
6. Enter distance how far to shift staffs and what direction
7. Image with staff shifted to desired amount should be found in demo directory
